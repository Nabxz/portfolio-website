Galaxy

Developed an interactive web application, "Galaxy," that serves as an educational and exploration tool for space enthusiasts. This platform integrates real-time astronomical data, leveraging APIs from NASA and Pixabay, to present a dynamic and informative user experience. Features include a captivating "Image of the Day" section, a comprehensive gallery of planetary images, live Earth events, and detailed weather information searchable by city or country. The site encourages user engagement through personalized event tracking with Google Calendar integration and offers in-depth explorations of planets and the solar system with rich visual content.

Key Features And Functionalities

  • Real-Time API Integration: Utilized the NASA API for fetching the "Image of the Day" and planetary images, and the Pixabay API for enhancing the site's visual content with high-quality images. This integration ensures the delivery of accurate and up-to-date space-related data and imagery.
  • Personalized User Accounts: Implements sign-in and account creation capabilities, allowing users to personalize their experience by saving favorite articles, managing a wishlist of interesting events, and customizing their exploration of the solar system.
  • Integration of Three.js and Solar System Library for Enhanced Web Experience: This project adeptly incorporates Three.js and a dedicated Solar System API to integrate a 3D solar system model into the web experience. This strategic combination enriches the website, offering users an engaging and visually stimulating exploration of celestial bodies. By employing these advanced technologies, the project exemplifies a seamless blend of data-driven content and interactive 3D graphics, showcasing the potential of modern web development to create compelling and informative user experiences.
  • Dynamic Content Presentation: Employs JavaScript functionalities for dynamic content filtering, showcasing articles and missions in both grid and list views. This adaptability enhances content accessibility and user engagement.
